## Execution

Each candidate must execute the project using the methodology provided as part
of the experiment.

The methodology is responsible for guiding the execution process.

The candidate must complete the project while respecting:

- the shared input;
- the project constraints;
- the acceptance criteria;
- the required deliverables.

The methodology may iterate, revisit or reorder work whenever justified.

The final repository must contain sufficient evidence to understand how the
project evolved from the initial input to the final handoff.
