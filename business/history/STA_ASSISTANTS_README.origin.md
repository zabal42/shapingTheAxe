# STA Assistants

**Estado:** exploración de producto  
**Versión documental:** 0.1  
**Propietario:** Mikel Zabal  
**Tecnología base:** ShapingTheAxe

## Propósito

STA Assistants es una especialización personal de ShapingTheAxe orientada a descubrir, modelar, definir y preparar asistentes de IA adaptados a negocios reales de forma repetible, trazable y verificable.

No es la empresa común de asistentes para pymes formada por Alberto, Pedro y Zabal.

## Pregunta central

¿Puede STA Assistants evolucionar desde una herramienta interna útil hasta un producto, consultoría o empresa tecnológica independiente y financiable?

## Hipótesis principal

STA Assistants puede reducir de forma demostrable el tiempo, las omisiones, los errores y el retrabajo necesarios para transformar el conocimiento de un negocio en una definición verificable y preparada para construir un asistente de IA personalizado.

## Estado actual

Todavía no se afirma que STA Assistants sea una startup, un SaaS, una consultoría, una plataforma o un producto financiable. Esas posibilidades deben validarse mediante evidencia.
